# 更新日志

本文件记录公开发行仓库面向用户的主要变化。

## 0.5.4 - 2026-07-22

- 仅作为源码/发行候选：Chrome Web Store v0.5.3 仍已发布；v0.5.4 尚未提交商店审核。v0.5.3 验证归档、校验和、图片和发行证据保持不可修改。
- 新增有边界的创作者研究、评论证据和搜索结果处理，使创作者身份、评论和搜索结论保留证据限制而非被推断。
- 加固扩展隐私 manifest 和公开打包检查，包括仅本地可执行代码与受限权限。
- 改进阶段遥测和清理报告，使清理失败可见而不覆盖主运行结果。
- 加固公开 `.env` 与 CI 合同：公开构建排除密钥和运行数据，并在 Windows、Ubuntu 校验发行边界。
- 发行证据不包含密钥、Cookie、平台原始载荷，也不包含公开商店条目引用以外的商店或账号标识。

## 0.5.3 - 2026-07-17

- 将公开产品统一命名为“ENHE 产品推广素材生成器 / ENHE Product Promo Maker”，并更新销售说明、产品链接和发行身份。
- 为 Chrome 扩展提供中英文界面与语言切换，保留当前标签页转任务、平台选择和本地命令生成路径。
- 完成扩展 11 项非支付命令与随包 Skill 脚本的本地功能同步审计；支付、订阅、许可证、点数和账单后端不纳入该同步结论，原有 billing UI 与 `billing-contract.json` 保留。
- 加强本机 MediaCrawler Sidecar 的固定上游版本、安装边界、运行锁、敏感输出清理、部分结果状态和平台验证/登录状态处理。
- 增加可独立构建、校验和发布的公共发行目录，包括中英文文档入口、组件清单、校验脚本、测试和确定性发行包流程。
- 明确 Hosted Worker 关闭；公开版使用本地 Skill 和本地生成命令，不把托管执行描述为可用能力。
- 明确最终发布由用户确认，Cookies 与 Chrome 登录配置只保留在本机，真实效果复盘只接收真实 URL、指标、评论、订单和收入证据。
